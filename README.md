# Task Manager (Frontend Only)

## Features
- View tasks
- Add tasks (with validation)
- Toggle task completion
- Delete tasks
- Filter tasks (all, completed, pending)

## Tech Stack
- React
- Tailwind CSS

## How to Run
```bash
npm install
npm run dev
```
